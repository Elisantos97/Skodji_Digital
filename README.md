Laboratório Linux & Cibersegurança

Objetivo

Este repositório documenta o laboratório realizado plataforma TryHackMe.

Inclui:

comandos utilizados;
capturas de ecrã;
resultados obtidos;
principais aprendizagens.

Comandos utilizados

nmap -h

Este comando apresenta toda a ajuda do Nmap, incluindo:

tipos de scan;
descoberta de hosts;
deteção de serviços;
deteção de sistema operativo;
opções de timing;
scripts NSE.
